## 🎯 CORE CAPABILITIES
 
### 1. NETWORK TOPOLOGY ANALYSIS
- **Real-Time Signal Logging:** Exhaustive Wi-Fi and Bluetooth detection engine with interactive heat mapping. **[IPHONE COMPATIBLE]**
- **OUI Categorization:** Automated identification of specific hardware categories (e.g., Fun-Watcher/Blue, Fun-Stopper/Red).
- **Topology Visualization:** Dynamic rendering of network structures and node relationships on the Grid dashboard. **[IPHONE COMPATIBLE]**
 
### 2. DEFENSIVE INTELLIGENCE
- **Tail Detection:** Multi-session encounter monitoring. Alerts after 3+ separate detections of the same device. **[IPHONE COMPATIBLE]**
- **Surge Alerts:** Automated notifications when a specific threshold of unique threat emitters (Fun-Stoppers) is detected. **[IPHONE COMPATIBLE]**
- **Hotspot Prioritization:** Identification of persistent cells based on historical presence (3+ days in a 7-day window). **[IPHONE COMPATIBLE]**
 
### 3. SERVICE ORCHESTRATION
- **Auto-Update Lifecycle:** Automated synchronization with GitHub releases to ensure all nodes run the latest security definitions.
- **Service Isolation:** Strict separation between the FastAPI scanning backend and the Vite/React management interface.
- **Hardware Integration:** Standardized support for RTL-SDR and HackRF signals into the topology graph.
 
### 4. PRIVACY & HARDENING
- **Identifier Masking:** HMAC-SHA256 hashing of all BSSID/BLE addresses before storage.
- **Local Secret Management:** Local generation of HMAC secrets at `%USERPROFILE%\Grid\.hmac_secret` to ensure zero-knowledge telemetry.

## 📱 IPHONE COMPATIBILITY SUMMARY
- **Compatible:** Topology Dashboard, Heat Map Visualization, Replay Mode, Alert Toasts (Surge/Tail), Hotspot Analytics, Management UI (Web/Mobile App).
- **Incompatible:** Native physical-layer Wi-Fi injection or direct raw Bluetooth scanning (requires desktop-class kernel permissions).
