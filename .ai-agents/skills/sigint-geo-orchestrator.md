# Skill: SIGINT Geo-Orchestrator
 
## Description
Spatial Intelligence & DeFlock blindspots. This skill provides advanced spatial reasoning for signal geolocation and DeFlock camera network mapping.
 
## Functions
- **map_blindspots(data):** Analyze camera coverage and identify unmonitored routes.
- **triangulate_signal(samples):** High-precision signal geolocation using TDOA/RSSI.
- **path_prediction(history):** Predict future movement paths based on past captures.
